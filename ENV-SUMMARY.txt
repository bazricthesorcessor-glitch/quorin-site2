TROUBLESHOOTING
================

Database Connection Error:
  - Check PostgreSQL is running
  - Verify DATABASE_URL in backend/.env
  - Run: bash scripts/setup-postgres.sh

CORS Error:
  - Update CORS_ORIGIN in backend/.env
  - Match your frontend URL
  - Restart backend

API Not Found:
  - Check backend is running
  - Verify VITE_API_URL in app/.env.local
  - Should be: http://localhost:3000/api

Port Already in Use:
  - Change PORT in backend/.env to 3001
  - Or stop other processes using port 3000
