# delete-me quarantine

This directory is a review area for files that appear unnecessary, obsolete or duplicated but should not be permanently deleted without confidence.

## Rules

- Preserve enough of the original path/name to understand where a quarantined file came from.
- Add a short reason when moving files here.
- Do not place active source files here.
- Bazric may permanently remove this directory's reviewed contents.

## Current quarantine

No files have been quarantined yet. The repository has not been fully audited for dead files, so nothing has been moved blindly.
