# Phase 1 — Backend Foundation

Objective:
Transform QUORIN from static frontend into a real application.

## Goals

- Create backend architecture
- Establish API communication
- Prepare scalable folder structure
- Enable future commerce logic

## Stack

Backend:
- Node.js
- Express.js
- TypeScript

Database:
- PostgreSQL

ORM:
- Prisma

## Core Principles

- simplicity first
- scalable later
- avoid premature complexity

## Initial Folder Structure

/backend
/src
/routes
/controllers
/services
/middleware
/prisma

## Initial APIs

GET /products
GET /products/:id
POST /inquiry

## Deliverables

- Express server running
- Prisma connected
- PostgreSQL connected
- Frontend successfully fetching products

Success Criteria:
Frontend displays live backend data successfully.
